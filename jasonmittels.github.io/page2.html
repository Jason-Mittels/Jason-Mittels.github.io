VI is a text editor written in 1976, for the original UNIX operating system, in C. It alllows coders to edit code very fast.
In '91, VI iMproved (vim) was released. It expanded on the original VI.
Vim is insanely large, I've read about decades-long vim users learning new and useful commands. I think we vim-learners want a list of the bare minimum amount of commands you need to know.
I have compiled such a list.

here are some shortcuts to get you started on your vim journey:

  • ZZ - save + quit
  • ZQ - quit
  • :w file - save as file
  • i - INSERT mode
  • a - move cursor one to the right, then switch to INSERT mode
  • t<char> - goto <char>
