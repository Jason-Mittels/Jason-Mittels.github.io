<--- most used vim shortcuts !--->

VI is a text editor written in 1976, for the original UNIX operating system, in C. It alllows coders to edit code very fast.
In '91, VI iMproved was released. Usually called VIM. It expanded on the original VI.

Here are some shortcuts to get you started on your vim journey:

  • ZZ - save + quit
  • ZQ - quit
  • :w file - save as file
  • i - INSERT mode
  • a - move cursor one to the right, then switch to INSERT mode
